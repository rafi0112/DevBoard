Question-1: What are the different ways to select an element in the DOM?

DOM elements are selected by ID (getElementById), class (getElementsByClassName), tag name (getElementsByTagName), or CSS selectors (querySelector, querySelectorAll).

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: Gets or sets HTML markup within an element, parsing tags.  
innerText: Gets or sets the visible text content, respecting rendered styles.  
textContent: Gets or sets all text content, including hidden elements.  

Question-3: What is event delegation in the DOM?

Event delegation is a technique in JavaScript where a single event listener is added to a parent element instead of adding multiple listeners to individual child elements. This works because of event bubbling, where an event triggered on a child element propagates up to its ancestors.

Question-4: What is event bubbling in the DOM?

Event bubbling is the process where an event starts from the target element and propagates upward through its ancestors in the DOM, allowing parent elements to handle child events.

Question-5: How do you create, add, and remove elements using JavaScript?

document.createElement() to create elements, appendChild() or insertBefore() to add them, and removeChild() or element.remove() to delete them dynamically from the DOM.
